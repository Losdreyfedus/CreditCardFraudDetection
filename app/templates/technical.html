Model Mimarisi

Sistemimiz, yüksek doğruluk ve düşük yanlış pozitif oranı sağlamak için Gradient Boosting algoritması kullanmaktadır. Model, dengeli veri seti üzerinde eğitilmiş ve çapraz doğrulama ile optimize edilmiştir. 


from sklearn.ensemble import GradientBoostingClassifier
from sklearn.preprocessing import StandardScaler

# Model parametreleri
model = GradientBoostingClassifier(
    n_estimators=100,
    learning_rate=0.1,
    max_depth=3,
    random_state=42
)

Veri Önişleme

Ham veriler, model performansını artırmak için bir dizi önişleme adımından geçirilir:

  • Standartlaştırma (StandardScaler)
  • Aykırı Değer Tespiti
  • SMOTE ile Veri Dengeleme
  • Özellik Mühendisliği

# Veri önişleme pipeline
from sklearn.pipeline import Pipeline
from imblearn.over_sampling import SMOTE

pipeline = Pipeline([
    ('scaler', StandardScaler()),
    ('sampler', SMOTE()),
    ('classifier', model)
])
